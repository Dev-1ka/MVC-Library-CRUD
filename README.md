# Library Management System - MVC CRUD Application

A simple ASP.NET MVC web application developed using the Code First approach and Scaffolding to perform CRUD (Create, Read, Update, Delete) operations for managing book inventory in a library system.

## Features

- Add new books
- View book inventory
- Update book details
- Delete books
- Database integration using Entity Framework
- MVC architecture implementation
- CRUD operations using Scaffolding

## Technologies Used

- ASP.NET MVC
- C#
- Entity Framework
- SQL Server
- HTML5
- Bootstrap

## Purpose

This project was developed to understand MVC architecture, Entity Framework Code First approach, database connectivity, and CRUD operations using Scaffolding in ASP.NET.
